# eSports System

Welcome to the **eSports System** project!  
This project is a modern web platform designed for **eSports enthusiasts, players, and event organizers**.  
It allows users to explore tournaments, watch live events, and manage competitive gaming content.

## Project Overview

The **eSports System** provides a seamless experience for gamers and event creators to connect through a single platform.  
Users can register, explore live events, manage content, and track ongoing eSports tournaments.

### Main Features:
- **User Login & Registration:** Secure access for both players and organizers.  
- **Live Events & Streams:** Browse and watch current or upcoming eSports tournaments.  
- **Event Pages:** Get detailed information, schedules, and live analytics for each event.  
- **Creator Dashboard:** Manage and upload new events or video content easily.  
- **Content Management:** Add highlights, replays, and event-related media.  
- **Modern Dark UI:** Sleek interface inspired by gaming aesthetics.

---


## Design Highlights

- Responsive, **dark-themed interface** designed for immersive user experience.  
- Clear content hierarchy with **neon color palette** inspired by eSports environments.  
- Modular layout for scalability and future feature extensions.  
- Consistent use of **typography, spacing, and iconography** for visual clarity.
