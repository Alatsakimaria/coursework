# Advanced C Programming - HY255
This directory contains the source code and implementations for the Advanced C module of my coursework. The focus of this project is on low-level memory management, efficient data structures, and mastering advanced C syntax.

## Contents

- ### MemoryManagement/: Exercises involving malloc, calloc, realloc, and free.

- ### DataStructures/: Custom implementations of Linked Lists, Trees, and Hash Maps.

- ### FileIO/: Advanced file handling and binary data processing.

- ### Pointers/: Deep dives into function pointers, pointer arithmetic, and multi-dimensional arrays.
